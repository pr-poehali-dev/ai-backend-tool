# QQRenta API v2 - Полная документация

## Базовый URL
https://api2.qqrenta.ru/api/v2

## Endpoint: Поиск жилья
**URL:** /search
**Метод:** GET

### Обязательные параметры:
1. **city** (string) - Название города для поиска
   - Примеры: "Москва", "Санкт-Петербург", "Казань", "Сочи"
   - Можно использовать русские или английские названия

2. **checkin** (string) - Дата заезда в формате YYYY-MM-DD
   - Формат: YYYY-MM-DD (год-месяц-день)
   - Примеры: "2025-11-15", "2025-12-01"
   - Важно: дата должна быть в будущем

3. **nights** (integer) - Количество ночей проживания
   - Минимум: 1 ночь
   - Примеры: 1, 3, 7, 14, 30

4. **guests** (integer) - Количество гостей
   - Минимум: 1 гость
   - Примеры: 1, 2, 4, 6

### Пример запроса:
```
GET https://api2.qqrenta.ru/api/v2/search?city=Москва&checkin=2025-11-15&nights=3&guests=2
```

### Формат ответа:
API возвращает JSON объект с массивом `results`, содержащим найденные объекты жилья.

### Структура объекта в results:
- **id** (number) - Уникальный ID объекта
- **title** (string) - Название объекта
- **full_address** (string) - Полный адрес
- **city** (string) - Город
- **price** (number) - Цена за ночь в рублях
- **bedrooms** (number) - Количество спален
- **guests** (number) - Максимальное количество гостей
- **photos** (array) - Массив фотографий
  - Каждое фото содержит: sm (маленькое), md (среднее), lg (большое)
- **external_reviews_rating** (number) - Рейтинг от 0 до 5
- **latitude** (string) - Широта
- **longitude** (string) - Долгота

### Примеры использования:

**1. Найти жильё в Москве на 15 ноября на 3 ночи для 2 гостей:**
```
city=Москва&checkin=2025-11-15&nights=3&guests=2
```

**2. Найти жильё в Сочи на неделю для семьи из 4 человек:**
```
city=Сочи&checkin=2025-12-20&nights=7&guests=4
```

**3. Найти жильё в Санкт-Петербурге на выходные:**
```
city=Санкт-Петербург&checkin=2025-11-08&nights=2&guests=2
```

### Важные замечания:
- API возвращает максимум 50 результатов
- Цены указаны за ночь
- Все даты в формате ISO 8601 (YYYY-MM-DD)
- Поддерживаются города России и некоторых стран СНГ
- Рейтинг от 0 до 5, где 5 - отлично

### Типичные ошибки:
1. Неверный формат даты - используйте YYYY-MM-DD
2. Дата в прошлом - выбирайте будущую дату
3. Некорректное название города - проверьте правописание
4. Отсутствие обязательных параметров

### Обработка ответа:
Успешный ответ содержит массив `results` с объектами жилья.
Каждый объект можно отобразить пользователю с фото, ценой, адресом и рейтингом.
